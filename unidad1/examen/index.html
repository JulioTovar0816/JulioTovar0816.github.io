Front-end

¿Que es el front-end?

Es la programacion del lado del cliente, que incluye solicitudes de datos y la interfaz que mira el usuario


Roles dentro del front-end

Diseñador UI: Diseña los flujos de usuario, pantallas e interacciones en un programa de diseño

Maquetador: Lleva el diseño a código HTML y CSS (No se ocupa de la lógica)

Programador Front-end: Agrega datos reales mediante un API y se encarga de la lógica

Principales lenguajes y frameworks


HTML, CSS, Javascript, Angular, React, Node JS

Back-end

¿Que es el back-end?

Este programador se encarga de la lógica que está en los sitios web, además de que maneja las bases de datos que estos utilizan


Roles dentro del back-end

Back-end developer: Se encarga de la lógica de negocio y crea las API´S para que el front-end pueda consumirlas

Administrador de BDD: Diseña, implementa, mejora y mantiene el sistema de BDD

Administrador del servidor: Gestiona la instalación, el soporte y mantiene el servidor donde se aloja el sitio web


Principales lenguajes y frameworks

GO, Java, PHP, Python, GraphQL


¿Qué es un stack?

Un conjunto de tecnologias que se pueden utilizar para resolver un problema
